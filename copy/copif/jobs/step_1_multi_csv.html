CSV Options

The CSV template contains the following columns. All of these columns must be included and each cell must contain either text or a numerical value in order to be processed successfully.

Name

Use the name column to give each job a relevant title
Words

The words column can specify a word count of between 100 & 100000 words
Brief

Provide a detailed brief with your exact requirements
Type

In the type column, select one of the following options:

  • Article
  • Press release
  • Web page
  • Blog
  • Review
Quality

In the quality column, specify either standard or professional copy
